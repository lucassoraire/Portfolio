PROYECTO : HOTEL NILUMA

Hotel Niluma es un sistema web completo para la gestión de un hotel boutique. Incluye frontend responsive, sistema de reservas, panel de administración y diseño de marca.

Diseño de interfaz

Se trabajó un diseño limpio con enfoque en UX. Cuatro secciones para que el cliente pueda conocer de manera simple las comodidades que brinda el Hotel, sus habitaciones como también las sucursales en las que se encuentran.Todo pensado para que la experiencia sea óptima.

Reservas funcionales

Implementación de sistema de reservas con PHP y MySQL. Cada cliente podía reservar su habitación luego de poder ver con detalle toda la información de la misma, seleccionaba fecha de ingreso y egreso, luego seleccionaba el método de pago y finalizaba la reserva. Todo simple para que no tenga que dar vueltas innecesarias.

Responsive design

Adaptado a dispositivos móviles para mejor experiencia. Desde la compu, desde la tablet o desde el celular, esta página se puede visitar de manera cómoda y legible con cualquier dispositivo.

Contacto

Siempre es bueno tener un apartado de contacto para que los clientes puedan ahí mandar sus consultas y expresarse de la mejor manera. Con solo pedir un mail o un teléfono el Hotel ya sabe que esta contacto con sus clientes.
